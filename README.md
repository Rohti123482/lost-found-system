# Universal Lost & Found Reporting System

A community-driven platform to report and locate missing pets and persons.

## Features

- Report lost pets or persons
- Report found pets or persons
- Interactive map visualization
- Distance-based filtering
- Automatic matching between lost and found reports

## Technology Stack

- Python
- Streamlit
- Supabase
- Folium
- Geopy

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
